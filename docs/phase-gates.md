# Phase Gates

Current phase: 1 - PROOF.

Open gate items:
- [ ] Reports understandable without explanation
- [ ] At least 50% meaningful actions per site
- [ ] Cost per site analysis is less than or equal to 0.10 USD
- [ ] Human reviewer finds output useful

Use `npm run gate:phase1` to generate the evidence needed for the objective gate items. The readability and usefulness checks require human review of generated reports.
